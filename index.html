What is Tenside UI?

Tenside is a web based package manager on top of Composer. You can also call it a "Web User Interface" for Composer.

Who (may) use Tenside?

Tenside is designed to work with all these systems. So it will work out of the box with all Composer ready projects. This also means, everyone who using Composer can use Tenside!

Why using Tenside?

There are many projects using Composer as their dependency and package manager. Not only frameworks like Symfony or Laravel, also CMS, eCommerce and user modular systems using Composer. Especially the last ones are our target group.

A user friendly extension management is the final goal of Tenside.
